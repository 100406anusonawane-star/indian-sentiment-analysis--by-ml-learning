# Indian Sentiment Analysis using Machine Learning

## Overview
This project performs sentiment analysis on Indian e-commerce product reviews.
It classifies reviews as **Positive** or **Negative** using Machine Learning techniques.

The project is built for learning purposes and demonstrates a complete ML pipeline.

---

## Dataset
- Indian product reviews (English + Hinglish)
- Two sentiment classes: Positive, Negative

Example:
- "Product bahut accha hai" → Positive  
- "Service bilkul bekar thi" → Negative  

---

## Methodology
1. Text preprocessing
2. TF-IDF feature extraction
3. Logistic Regression model
4. Model evaluation
5. Prediction on new reviews

---

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP techniques
